Notice | Default

By default upsell notice doesn't have any spacing. Instead, it takes it from the parent element as in this example it takes it from sui-box-body.

Notice | Modified

In case you need to add spacing to the element you must use utility classes like:

Class Description
sui-padding It adds 30px padding on desktop and 20px on mobiles respecting library default gutter.
sui-padding--hidden-mobile This modifier removes padding from mobiles only.
sui-padding--hidden-desktop This modifier removes padding from desktops only.
sui-padding-${position} It adds to (top/bottom/left/right) position a padding of 30px on desktop and 20px on mobiles respecting library default gutter.
sui-padding-${position}--hidden-mobile This modifier removes from (top/bottom/left/right) position the assigned padding on mobiles only.
sui-padding-${position}--hidden-desktop This modifier removes from (top/bottom/left/right) position the assigned padding on desktop only.
Reminder:
Don't forget to replace ${position} with the desired position (top/bottom/left/right). For example, sui-padding-top.
